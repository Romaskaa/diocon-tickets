# Управление задачами

| Статус | Значение | Когда используется | Видимость | Кто может перемещать |
|--------|----------|--------------------|-----------|----------------------|
| BACKLOG | Задача в общем списке работ, не готова к выполнению | Идеи, пожелания, задачи на будущее, задачи, требующие уточнения | Видна всем участникам тикета/проекта | Support Manager, Task Owner |
| TODO | Задача готова к выполнению, взята в спринт/очередь | Чётко сформулирована, есть критерии приёмки, можно начинать работу | Видна исполнителям | Support Manager, Assignee, Task Owner |
