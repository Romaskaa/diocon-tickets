# Правовая модель работы с договорами

## Новые системные роли

 - `ACCOUNT_MANAGER` - Менеджер по работе с клиентами
 - `FINANCE` - Финансовый специалист / Бухгалтер

## Расширение ролевой модели

| Роль | Описание | Уровень доступа к Contracts | Комментарий |
|------|----------|-----------------------------|--------------|
| ACCOUNT_MANAGER | Менеджер по работе с клиентами | Полный | Новая ключевая роль |
| SUPPORT_MANAGER | Руководитель поддержки | Полный / ограниченный | Уже есть |
| SUPPORT_AGENT | Агент поддержки | Ограниченный (только просмотр) | Уже есть |
| FINANCE | Финансовый специалист / Бухгалтер | Полный (только чтение + отчёты) | Новая роль |
| ADMIN | Системный администратор | Полный | Уже есть |
| CUSTOMER | Обычный клиент | Только свои договоры | Уже есть |
| CUSTOMER_ADMIN | Администратор клиента | Свои договоры | Уже есть |

## Ролевая матрица прав

| Действие                                  | ACCOUNT_MANAGER | SUPPORT_MANAGER | SUPPORT_AGENT | FINANCE | ADMIN | CUSTOMER | CUSTOMER_ADMIN |
|-------------------------------------------|-----------------|-----------------|---------------|---------|-------|----------|----------------|
| Создание нового договора                  | Да              | Нет             | Нет           | Нет     | Да    | Нет      | Нет            |
| Редактирование договора                   | Да              | Ограниченно     | Нет           | Нет     | Да    | Нет      | Нет            |
| Добавление/изменение пакетов часов        | Да              | Нет             | Нет           | Нет     | Да    | Нет      | Нет            |
| Просмотр остатка часов по своим договорам | Да              | Да              | Да (свои)     | Да      | Да    | Да       | Да             |
| Просмотр всех договоров компании          | Да              | Да              | Нет           | Да      | Да    | Нет      | Нет            |
| Списание часов (при закрытии тикета)      | Нет             | Да              | Да            | Нет     | Да    | Нет      | Нет            |
| Одобрение TimeEntry (фактических часов)   | Нет             | Да              | Нет           | Да      | Да    | Нет      | Нет            |
| Генерация отчётов по утилизации часов     | Да              | Да              | Нет           | Да      | Да    | Нет      | Нет            |
| Приостановка / возобновление договора     | Да              | Нет             | Нет           | Да      | Да    | Нет      | Нет            |

## Кто и как работает с договорами

Вот запрошенная таблица в формате Markdown:

| Роль            | Основные обязанности                                        | Как работает с системой                                            |
|-----------------|-------------------------------------------------------------|--------------------------------------------------------------------|
| ACCOUNT_MANAGER | Заключение и ведение договоров, контроль остатка, продление | Создаёт ServiceContract, добавляет пакеты, отслеживает потребление |
| SUPPORT_MANAGER | Контроль выполнения договоров, распределение нагрузки       | Просматривает остаток, одобряет TimeEntry, может приостанавливать  |
| SUPPORT_AGENT   | Выполнение работ по тикетам                                 | Видит остаток по своим тикетам/проектам, создаёт TimeEntry         |
| FINANCE         | Финансовый контроль, биллинг                                | Просматривает потребление, готовит отчёты, сверяет с оплатой       |
| CUSTOMER_ADMIN  | Контроль своих договоров со стороны клиента                 | Просматривает остаток, видит историю списаний                      |
